Kube-Linter Binary
=====

Installs [Kube-Linter Binary](https://docs.kubelinter.io/)

Dependencies
------------

None.

Role Variables
--------------

All variables which can be overwritten are stored in [vars/main.yml](vars/main.yml)

Example Playbook
----------------

```yml
- hosts: servers
  roles:
    - app-kubelint
```
